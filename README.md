# GraduateReport

成都理工大学图书馆为毕业生准备了纪念毕业的明信片。

本程序根据毕业生的一卡通号，从成都理工大学图书馆WebApi中调用数据。

根据程序提供的Word模板，为该毕业生创建其在图书馆中的一些有纪念意义的数据。

程序用读取的数据填充Word模板数据,填充完毕后，保存成一个新的Word文档，然后调用SHELL打开该Word文档。

接下来，工作人员手动将该Word文档打印到明信片背后。

在程序运行时，如果Word模板不存在，则自动创建一个模板。

工作人员可以通过修改Word模板的方式，来确定最终的Word文件内容呈现效果。


具体填充方法：
遍历第一个TABLE，读取TABLE中第一列中的文本，根据该文本内容，在获取的WebApi数据中查找对应的项，获取其值，然后填充第二列。

使用的第三方包：
CommunityToolkit.Mvvm
FreeSpire.Doc
